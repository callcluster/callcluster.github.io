Callcluster

Tool for callgraph extraction, analysis and visualization

Demo

Objetivo

Diseñar e implementar un conjunto de herramientas de extracción, visualización y análisis de callgraphs para facilitar la evaluación y mejora de la arquitectura de una aplicación programada en C o C#.

Manuales de usuario

Callcluster está compuesto de 3 programas independientes:

  • Extractor Roslyn: extrae callgraph y estructura de un programa C#
  • Extractor clang: extrae callgraph y estructura de un programa C
  • Visualizador: Permite visualizar los datos extraídos

Documentación técnica

  • descripción del formato de analysis.json. Podés usarlo para crear tus propios extractores para cualquier lenguaje de programación junto con callcluster-visu.
